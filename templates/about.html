About CianTube

CianTube has been made by me, Mary-Ann Horley, as a final project for an MA in Digital Media at the University of Lincoln. It is not associated with YouTube or Google. It is named after my five-year-old son Cian.

Principles

  • Safe viewing
  • Minimal data collection
  • Inspiration not “engagement” AKA addiction

Credits

This site relies on:

  • The Python language, specifically Flask 0.12 for the basic structure, hosted on AWS Lambda with the help of Zappa
  • Python modules PyMongo, Furl, Requests, NLTK, TextBlob, WTF
  • Bootstrap 4 for basic interface elements
  • FontAwesome for icons
  • The font Fredoka One via Google Fonts
  • MongoDB 3.4 on MLab to store data
  • The YouTube Data API v3 to gather videos
  • Keen.io and its Python module to collect anonymous usage data
  • The endless patience of Laury, Cian and Penny!
